## 🔐 Sistema de Autenticação

O sistema possui autenticação de usuários, permitindo que cada usuário tenha seu próprio histórico de ideias geradas.

Funcionalidades:

* Cadastro de usuário
* Login
* Sessão autenticada
* Histórico de ideias salvo por usuário

---

## 🗄️ Banco de Dados

As informações são armazenadas em banco de dados, incluindo:

* Usuários
* Ideias geradas
* Histórico de conversas
* Data de criação das ideias

---

## 🧠 Geração de Ideias com IA

A IA gera ideias completas contendo:

* Nome do projeto
* Descrição
* Público-alvo
* Funcionalidades
* Monetização
* Tecnologias recomendadas
